# web-client

React + Vite + TypeScript + Tailwind frontend.

## Run

    pnpm install
    pnpm dev

Dev server: http://localhost:5173

## Test

    pnpm test

## Lint / format

    pnpm lint
    pnpm format

## Regenerate API types from the spec

    pnpm gen
    # or from repo root: ../api/scripts/gen-all.sh
